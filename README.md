# 3D-Character-Projection-using-Kinect-V2-and-Unity3D
A Kinect V2 captures the user's body joints as they stand in front of it and maps these movements onto a 3D character, enabling the character to replicate the user's actions in real-time. Using Unity3D, a concert stage environment is created for the character’s performance. Multiple C# scripts are implemented to handle the motion mapping, and the Kinect V2 package is imported into Unity3D to build the character’s skeleton structure.
